# Telegram Voice Transcribator (`transcribator`)

Telegram-бот для перевода голосовых сообщений в структурированный текст. Применяет технологию распознавания речи OpenAI Whisper и интеллектуальное редактирование текста на базе больших языковых моделей (LLM) от Groq Cloud с поддержкой пользовательских инструкций и сохранением настроек в PostgreSQL.

---

## 🎙️ Основные возможности

### 1. 📝 Высокоточное распознавание речи
* Перевод входящих голосовых сообщений (в формате `.ogg`) в текст с помощью **OpenAI Whisper-1 API**.

### 2. ⚡ Интеллектуальные режимы постобработки
После распознавания текст обрабатывается моделью **Llama-3.3-70b-versatile (Groq)** согласно выбранному режиму:
* **📝 Режим «Транскрипция»**: Добавляет правильную пунктуацию, исправляет орфографические ошибки, сохраняя эмоции автора и оригинальный темп речи.
* **✨ Режим «Косметические изменения»**: Очищает речь от междометий и слов-паразитов, разбивает на читаемые абзацы, выстраивает деловой, конструктивный слог в стиле **инфостиля Максима Ильяхова**.

### 3. 🎯 Режим «Свой промпт» (Custom Prompts)
* Пользователи могут создавать собственные правила обработки текста (до 3 активных промптов одновременно).
* Например: *«Переведи этот текст на английский»*, *«Сделай краткую выжимку (Summary)»* или *«Перепиши текст в виде списка задач»*.
* Возможность удобно переключаться, добавлять и удалять кастомные промпты через интерактивные кнопки.

### 4. 🗄️ База данных PostgreSQL
* Сохранение выбранного режима, незавершенных действий меню (FSM-подобное состояние) и списка созданных пользователем промптов в формате JSONB в СУБД **PostgreSQL**.

### 5. 📎 Обработка длинных текстов
* При транскрибации больших аудиофайлов бот автоматически разделяет результат на части, не превышающие лимит Telegram в 4096 символов.

---

## 🛠️ Технологический стек

* **Язык**: Python 3.10+
* **Фреймворк**: `python-telegram-bot` (версия 21.0+)
* **База данных**: PostgreSQL (библиотека `psycopg2`)
* **API распознавания**: OpenAI API (модель Whisper-1)
* **API обработки**: Groq Cloud API (модель Llama-3.3-70b)
* **Сеть**: `httpx` (асинхронные запросы)

## ⚙️ Настройка и Запуск

### 1. Переменные окружения
Создайте `.env` файл или укажите переменные окружения:

```env
TELEGRAM_BOT_TOKEN=your_telegram_bot_token
OPENAI_API_KEY=your_openai_api_key
GROQ_API_KEY=your_groq_api_key
DATABASE_URL=postgresql://user:password@host:port/dbname
```

### 2. Подготовка базы данных
Бот автоматически инициализирует таблицу `user_data` в PostgreSQL при первом старте.

### 3. Установка зависимостей и запуск
```bash
# Установка библиотек
pip install -r requirements.txt

# Запуск бота
python bot.py
```
